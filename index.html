진로검사 결과에서 사업계획서까지 이어지는 교육용 AI 에이전트

직업계고 학생이 진로검사 결과를 이해하고, 자신의 관심 분야와 연결된 창업 아이템을 구체화한 뒤 시장 분석, 비즈니스모델, 사업계획서 초안까지 단계적으로 완성하도록 돕는 프로젝트입니다.

5단계 AG01부터 AG05까지 연결된 에이전트 흐름
6종 공공데이터 수집 검증 완료 데이터소스
2026.07.13 현재 진행상황 기준일

학습 흐름

AG01
진로분석진로검사 PDF를 구조화된 프로필로 변환
AG02
창업아이템 추천진로 프로필과 연결된 아이템 3개 제안
AG03
시장·고객 분석고객 문제와 시장 근거 점검
AG04
비즈니스모델 설계가치제안, 수익, 운영 구조 정리
AG05
사업계획서 작성JSON과 Markdown 초안 및 피드백 생성

에이전트 구성

AG01

진로분석 에이전트

학생 자료를 익명 프로필 JSON으로 정리하고, 확인되지 않은 정보는 추측하지 않습니다.

AG02

창업아이템 추천

진로분석 결과를 바탕으로 수업에서 다룰 수 있는 창업 아이템 후보를 제안합니다.

AG03

시장·고객 분석

선택된 아이템의 고객, 문제, 경쟁 대안, 공개 데이터 근거를 정리합니다.

AG04

비즈니스모델 설계

가치제안, 핵심 기능, 수익·비용 가설과 검증 질문을 구성합니다.

AG05

사업계획서 작성·피드백

앞 단계 결과를 사업계획서 초안으로 만들고 개인정보 형태를 가린 뒤 피드백합니다.

현재 진행상황

  • ✓AG01~AG05 Skill, 프롬프트, 표준 JSON, 샘플 데이터 1차 구현 완료
  • ✓학생 제출, AG01 처리, 개인정보 검증, 중복 방지, 교수자 조회 흐름 스모크 테스트 통과
  • ✓AG01 JSON 기반 AG02~AG05 JSON·Markdown 전체 파이프라인 테스트 통과
  • ✓기업마당, 상권, KOSIS, OPENDART, 네이버 데이터랩, KIPRIS 수집 검증 완료

개인정보 보호 기준

  • ✓학생 이름 대신 익명 `student_profile_id` 사용
  • ✓원본 PDF와 결과 JSON 분리
  • ✓외부 AI 전송 전 학교 승인, 비식별화, 최소 정보 전송 원칙 적용
